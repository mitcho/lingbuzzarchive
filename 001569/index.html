Chomsky's Foundational Admission
Paul Postal
July 2012
 

This paper analyzes recent remarks by Noam Chomsky in effect admitting that his biolinguistics framework is incoherent. This is done in the context of a brief discussion of the incoherence independent of the admission, documenting various contradictions in Chomsky's foundational writings. It briefly draws some ethical and sociological conclusions.
