The universal base hypothesis: VO or OV?
Hans Broekhuis
May 2006
 

The universal base hypothesis postulates that all languages have the same underlying word order, which is advantageous because it simplifies language acquisition. According to Kaynes (1994) LCA, this order is VO so that the OV-languages must be derived by leftward movement of various types of VP-internal constituents: DP-objects, PP-objects, predicative phrases, etc. Haider (1997a) and Barbiers (2000) propose an underlying OV-order and claim that this gives rise to a simpler grammar since the VO-languages can be derived by means of a single operation, namely V-movement across the VP-internal constituents. This paper will argue that this simplification is only apparent.
