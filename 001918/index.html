Gradient and Iconic Features in ASL [Squib]
Philippe Schlenker
February 2015
 

We argue that the height specifications of ASL loci can display a highly iconic and gradient behavior while simultaneously sharing a key grammatical property of phi-features in spoken languages. We take this result to be consistent with the view that Universal Grammar allows for grammatical features with a gradient and iconic semantics.
