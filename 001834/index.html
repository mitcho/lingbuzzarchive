Antisymmetry and Hixkaryana
Michael Barrie
July 2011
 

This paper proposes an analysis for the basic clause structure of Hixkaryana, an OVS language, within an Antisymmetric framework. There are two important contributions to the study of OVS languages and Antisymmetry contained herein. First, given the universal order, Specifier-Head-Complement, it is proposed the OVS order arises by smuggling the object across the subject by raising the VP containing the object across the subject. Second, I propose a new generalization of OVS languages that is captured by the Antisymmetric analysis put forward. Namely, I show that, while OXV is quite widespread in SOV languages, it is absent in OVS languages. This is because SOV languages allow significantly more freedom in object placement than do OVS languages since the object must remain sufficiently low in order to be fronted along with the VP. This asymmetry would remain mysterious under a symmetric framework.
