Linguistic Missing Links.
Gerard Cheshire
January 2018
 

Linguistic missing links: instruction in decrypting, translating and transliterating the only document known to use both proto-Romance language and proto-Italic symbols for its writing system. This paper provides the solution to understanding the hitherto unknown writing system used for the manuscript listed as MS 408 at the Beinecke Library, Yale University. The writing system uses symbols, punctuation, grammar and language that are each unique. The manuscript is not encrypted, in the sense that its author made an effort to conceal the contents of the manuscript, as has been presumed by some scholars. Instead, it is code only in the sense that the modern reader needs to be versed in the calligraphic and linguistic rules to be able to translate and read the texts. Furthermore, in discovering its writing system, it became apparent that the manuscript is of invaluable importance to the study of the evolution of the Romance languages and the scheme of Italic letters and associated punctuation marks now commonplace in those and other modern languages. This version of the paper contains further evidence of proto-Romance, as the zodiac pages have the names of the months overwritten with transitional Italics (Medieval modern), which can be easily located in the various modern Romance languages. Thus, we have proto-Romance words using both proto-Italica and modern Italics in the same manuscript.
