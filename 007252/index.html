A handout handout
David Pesetsky
July 2020
 

In the pandemic summer of 2020, Athulya Aravind and David Pesetsky gave a short online professional development class for MIT graduate students — which included discussion of handout creation (among other topics). This is a handout about handouts created for that class. Enough people have found it helpful since then that it it might be useful to make it more widely available, so here it is.
