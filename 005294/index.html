Immobile wh-phrases in Tagalog
Norvin Richards
July 2020
 

Offers an argument that "overt movement" cannot be understood as being triggered by a need to create a specifier for the head containing the Probe; rather, overt movement is driven by a need to make the Probe and the Goal linearly close to each other. The argument comes from Tagalog, which has obligatory overt wh-movement for many speakers. It is a predicate-initial language, and predicates can be wh-expressions (for example, in translations of questions like Where is Mary?). I argue that when a predicate is a wh-expression, it is in the structural position that predicates ordinarily occupy, and not in the (structurally higher) position in which wh-movement generally lands. The landing sites of overt wh-movement in Tagalog, then, are structurally diverse, but can be described straightforwardly in linear terms; Tagalog wh-phrases must be linearly adjacent to the complementizer.
