Generalized Earliness
Norvin Richards
April 2023
 

Much current work in syntax posits a derivation in which the 'narrow syntax' operates on representations which lack any phonological or semantic information, creating purely syntactic representations which are sent to interfaces which perform only those operations which are relevant for PF or for LF. I propose replacing this with a model in which operations of all kinds are in principle interspersed, regulated only by a version of Pesetsky's (1989) Earliness: every operation is constrained to occur as early as it can. I claim that once computational operations are stated correctly, Earliness will correctly capture the apparent independence between syntactic, phonological, and semantic modules, while still allowing those modules to interact in certain ways which I argue are desirable.
