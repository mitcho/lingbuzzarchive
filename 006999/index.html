The Imagistic Interface as a Possible Component of the Broad Language Faculty (FLB)
Nishant Mulay
December 2022
 

This paper talks about the possible existence of another interface within the broad language faculty (FLB) -- the Imagistic Interface -- and sketches a path of how one could study this interface using theoretical tools of contemporary internalist cognitive sciences.
