Natural Languages Are Not Generative Systems 2023
Paul Postal
April 2023
 

A pillar of the generative grammar movement is a technical view about the nature of systems which can characterize NLs, grammars. This takes them to be what in logical terms are. proof-theoretical devices. In linguistics these are called generative grammars. It is now repeatedly claimed that some kind of necessity requires that NL grammars be generative devices.The present work argues at length that not only is there no necessity for NL grammars to be generative devices, at least for any NL sharing the properties documentable for English, no generative grammar is even possible. It is shown that the dominant view about the role of generative grammars simply ignores a vast range of types of NL sentence which cannot be described in generative terms. It is pointed out that there is a clear alternative to generative devices, so called model-theoretic systems, which are compatible with the range of ignored sentence types beyond generative description.
