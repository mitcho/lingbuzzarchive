A'ingae language documentation
Justin Bai, Maksymilian Dąbkowski, Kalinda Pride, Nicholas Tomlin
August 2017
 

A'ingae is an underdocumented language isolate spoken in Ecuador and Colombia, with approximately 1500 speakers. This project involves working with native speakers to collect texts in A'ingae, perform linguistic analysis, and build web/mobile interfaces to display these texts.
