Priority union and feature logic in phonology
Charles Reiss
August 2020
 

Rules built from a single operation, priority union, can model both feature-filling and feature-changing processes. The distinction is handled by specifying which argument of priority union is defeasible. Priority union should be considered as a candidate for inclusion in phonological Universal Grammar.
