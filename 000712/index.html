Information Structuring Begins with The Numeration
Enoch Aboh
June 2008
 

This paper argues for the strong minimalist view that Information Structure starts in the numeration in the form of discourse-related lexical items which drive the derivation. In many languages, these discourse-related particles, which encode among other features Interrogative force, Topic, or Focus, show very specific syntactic behavior indicating that they project in syntax. The analysis further shows that these features are comparable to other optional formal features (e.g., Case, phi-features) that are added arbitrarily when the lexical item enters the numeration.
