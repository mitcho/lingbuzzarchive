Wh-agreement and bounded unbounded movement
Patricia Schneider-Zioga
July 2009
 

Contra the main stream analyses of wh-agreement, Kinande is argued not to have successive cyclic movement, despite the fact that it leaves familiar morphological marking on each relevant cycle (wh-agreement). The main evidence in support of this analysis is that Kinande does not exhibit reconstruction effects with A'-movement, which suggests that it involves a resumptive strategy to accomplish long distance displacement. This is because, in effect, all clauses, embedded or not, act like main clauses, which results in systematic islands. As a consequence, wh-agreement in Kinande and presumably in other languages with wh-agreement, without checking for reconstruction effects, cannot be taken as evidence for feature motivated intermediate checking of links of an A’-movement chain, nor can it be taken as evidence of successive cyclic movement.
