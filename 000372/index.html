Partially ordered categories: Optionality, scope, and licensing
Raffaella Bernardi, Anna Szabolcsi
November 2007
 

This paper uses a partially ordered set of syntactic categories to accommodate optionality and licensing in natural language syntax. A complex but well-studied data set pertaining to the syntax of quantifier scope and negative polarity licensing in Hungarian is used to illustrate the proposal. The presentation is geared towards both linguists and logicians. The main ideas can be implemented in different grammar formalisms.
