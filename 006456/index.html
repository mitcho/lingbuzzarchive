Plastics
Charles Reiss
October 2021
 

Science, including linguistic science, is never about the attested data. If chemists had only been interested in attested elements and compounds, we wouldn't have plastics or synthetic elements like copernicium. As soon as one analyzes languages in terms of a base of features, syntactic categories, rule orderings, and so on, simple combinatorics gives us an intensionally defined set of languages for which each particular language has an infinitesimal probability of attestation. The object of study for linguistics is that base of categories, that set of universal building blocks that comprise all individual languages.
