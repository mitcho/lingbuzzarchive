Rethinking French Dative Clitics (in light of frozen scope effects)
Dominique Sportiche
January 2020
 

English has both a prepositional dative constructions (PDC) and a double object construction (DOC) with different properties.1 One distinguishing property is the frozen scope effect only found in the DOC. First, this note documents that this effect is sometimes found in French too and concludes that French, like English, has a distinction between PDCs and DOCs, as suggested in Anagnostopoulou (2005). It next discusses the fact that this effect is only found in the presence of a dative clitic, suggesting that Dative clitics are only available for IOs in DOC constructions and not in PDC constructions and discusses how DOCs surface in French and concludes they do not.
