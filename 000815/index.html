Silent heads
Halldor Armann Sigurdsson, Joan Maling
January 2012
 

This paper takes as its starting point the suggestion of Cinque (1999:127) that the entire array of functional projections  [is] present in every sentence. We refer to this suggestion as CINQUES GENERALIZATION, extending it by arguing that syntax contains active (probing) silent heads, as well as inactive ones. It follows that there are no one-to-one mappings from syntax onto morphology, syntax being more abstract than commonly assumed.
