Tsez Syntax: A Description
Maria Polinsky
July 2015
 

This is a description of Tsez syntax, created as part of the volume “A Grammar of Tsez” prepared by Bernard Comrie and myself. Because this work is still a draft, chapters are not numbered, and each chapter has its own example numbering. I am posting it now in hopes of getting comments, criticisms, and suggestions before the description is sent to the publishers.
