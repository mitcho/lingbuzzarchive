Inverse Marking in Plains Cree
Yuriy Kushnir
June 2015
 

In this work, I argue that the so-called inverse agreement pattern observed in the Algonquian language Plains Cree is, in fact, a system displaying the more common typological feature called dierential case marking, inuenced by an intrinsic salience hierarchy built into the grammar. By re-analyzing the direction markers of Plains Cree as case markers, I assert that there is no (stipulative) morpho-syntactic feature of the type {INV} as assumed in previous work.
